## Iniciar base de datos

1. Clonar proyecto
2. Instalar paquetes
```
$ yarn install
```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Modificar las variables de entorno
5. Iniciar base de datos
```bash
$ docker-compose up
```
6. Iniciar Nest en modo dev
```
$ yarn start:dev
```
